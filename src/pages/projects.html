Batalha de Ninjas Naruto

SPRING BOOT, JAVA, MYSQL, SWAGGER - OPEN API

Este projeto implementa um CRUD (Create, Read, Update, Delete) completo para gerenciar uma emocionante 'Batalha de Ninjas' inspirada no universo Naruto. Usuários podem cadastrar novos ninjas, definir suas habilidades e participar de combates dinâmicos, acompanhando o desempenho e a evolução de cada personagem.
